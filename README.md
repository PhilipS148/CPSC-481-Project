# Agent Setup/Run Instructions

1. To run the game, make sure to have Pygame installed, and of course Python
2. Run the main file in the termial: 'python main.py'
3. Once the game is running, press the 'a' key to toggle the AI agent on/off. Manual play can be done with the arrow keys.
* To change weights for viewable agent play, change the 'DEFAULT_WEIGHTS' paramter on line 26 of main.py, to any preset from 'config.py'

# AI Benchmark Instructions

1. To run benchmarks: 'python ai_benchmark.py' in the terminal
* Heuristic weight values can be changed in 'config.py'
* To change weights for benchmarks, change the 'DEFAULT_WEIGHTS' paramter on line 31 of ai_benchmark.py, to any preset from 'config.py'
* The amount of trials AND the number of runs per trial done can be changed on line 178 of ai_benchmark.py, in case you want a shorter or longer one
